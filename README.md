# gymtrack

A terminal-based workout tracker written in C

Works on Linux and macOS

## Features
  - Create seperate routines (workouts) with different exercises
  - Track each exercises' sets and reps
  - View routine statistics and exercise progressions
  - Keyboard oriented

## Prerequisites
  - `make`
  - `gcc`
  - `ncurses` library

## Installation
Clone this repository:
```
git clone https://github.com/MicasiO/gymtrack.git
cd gymtrack
```
Install (might require sudo):
```
make install
```
Run:
```
gymtrack
```

## Uninstall
Change to cloned directory, then:
```
make uninstall
```
